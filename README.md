# AURA — Sistema de Gestão de Ativos e Manutenção

O **AURA** é uma solução completa para gestão de ativos, controle de ordens de serviço (OS) e suporte à decisão (BI) voltada para o setor de manutenção industrial/predial.

## 🚀 Funcionalidades Principais

* **Gestão de Ativos:** Cadastro e rastreamento de equipamentos e ambientes.
* **Controle de Manutenção:** Abertura e acompanhamento de chamados e ordens de serviço.
* **BI & Performance:** Painel de indicadores (Dashboards) com:
    * Taxa de Conclusão de OS.
    * Custo total acumulado (peças + mão de obra).
    * Tempo Médio de Atendimento (TMA).
    * Análise de recorrência de falhas por ativo.
* **Relatórios:** Exportação automatizada de relatórios em PDF para auditoria técnica.
* **Controle de Usuários:** Gestão de acessos ao sistema.

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido com as seguintes ferramentas:

* **Flutter:** Framework para interface do usuário multiplataforma.
* **GetIt:** Injeção de dependência para organização do código.
* **FL Chart:** Visualização de dados e gráficos interativos.
* **Printing & PDF:** Geração e impressão de documentos técnicos.
* **Dart:** Linguagem principal do projeto.
